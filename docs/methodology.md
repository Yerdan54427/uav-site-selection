# 方法说明

本文档说明本项目在校园无人机起降点选址分析中使用的主要方法，包括数据组织方式、评分方法、加权计算方法和可视化方法。

## 1. 研究思路

本项目采用“人工评分 + 加权综合评价 + 可视化对比”的方法，对多个候选起降点进行分析。

整体流程如下：

1. 人工确定候选起降点
2. 根据统一评分细则对各候选点进行人工评分
3. 将评分结果整理为 CSV 数据表
4. 使用加权总分模型计算综合得分
5. 通过图表展示各候选点的指标表现和最终排名

这种方法的优点是结构清晰、实现简单，适合课程项目、方案比较和初步选址分析。

## 2. 数据组织方法

本项目使用 `data/candidate_scores.csv` 作为输入数据文件。CSV 中每一行表示一个候选点，每一列表示一个评价指标。

数据表包含以下字段：

- `candidate`：候选点名称
- `service_distance`：服务区距离评分
- `logistics_distance`：物流中心距离评分
- `openness`：场地开阔度评分
- `obstacle_risk`：障碍物风险评分
- `crowd_risk`：人流风险评分
- `route_access`：航线通达性评分
- `operation_convenience`：运维便利性评分

其中，除 `candidate` 外，其余字段均为评分列。

## 3. 人工评分方法

本项目采用人工实地判断或基于场地信息的主观评价方式，对每个候选点在 7 个指标上进行打分。

评分统一使用 **1 到 5 分制**：

- `5` 分：表现优秀
- `4` 分：表现较好
- `3` 分：表现一般
- `2` 分：表现较差
- `1` 分：表现很差

评分细则已经在 [scoring_rules.md](/C:/Users/Yerdan/Desktop/uav-site-selection/docs/scoring_rules.md) 中单独给出。

这种方法适合当前项目规模，能够把场地条件、运行风险和服务效率等因素统一整理成可比较的数据。

## 4. 加权综合评价方法

为了避免只看单一指标，本项目使用加权总分法对候选点进行综合评价。

各指标权重如下：

- `service_distance`：`0.15`
- `logistics_distance`：`0.15`
- `openness`：`0.20`
- `obstacle_risk`：`0.15`
- `crowd_risk`：`0.15`
- `route_access`：`0.10`
- `operation_convenience`：`0.10`

综合得分计算公式为：

```text
total_score = 各指标分数 × 对应权重 后求和
```

权重设计体现了本项目的分析重点：

- 场地开阔度权重最高，因为它对安全起降影响较大
- 服务区距离和物流中心距离反映了服务效率与配送衔接效率
- 障碍物风险与人流风险体现运行安全性
- 航线通达性和运维便利性体现实际部署与长期运行的可行性

## 5. 数据校验方法

为了保证分析结果可靠，程序在读取 CSV 后会先进行基础校验，包括：

- 是否存在 `candidate` 列
- 是否包含全部 7 个评分指标列
- 各评分是否为数字
- 各评分是否都在 `1` 到 `5` 范围内

如果数据不符合要求，程序会直接报错并停止运行，避免异常数据影响最终结果。

## 6. 排序方法

程序会在计算出 `total_score` 后，按照总分从高到低对候选点进行排序。

排序结果会：

- 输出到终端
- 用于生成柱状图排名
- 用于选取前两名候选点绘制雷达图

这样可以保证文本结果和图表结果保持一致。

## 7. 可视化方法

本项目使用 `matplotlib` 生成 3 类图表，用于从不同角度展示选址分析结果。

### 7.1 评分热力图

热力图用于展示所有候选点在各个评价指标上的原始得分情况。

作用是：

- 观察不同候选点在单项指标上的差异
- 快速识别优势项和短板项
- 从整体上比较评分分布

### 7.2 综合评分柱状图

柱状图用于展示所有候选点的最终加权总分。

作用是：

- 直观展示最终排名
- 便于比较候选点之间的综合差距
- 作为最终推荐结果的重要依据

### 7.3 前两名雷达图

雷达图用于比较综合得分最高的两个候选点在 7 个指标上的表现。

作用是：

- 对比前两名候选点的结构性差异
- 识别各自的强项与弱项
- 帮助解释为什么某一候选点排名更高

## 8. 程序实现方法

本项目使用 Python 实现，主要依赖：

- `pandas`：用于读取 CSV、处理表格数据、计算加权总分
- `matplotlib`：用于生成热力图、柱状图和雷达图

程序入口为 `src/main.py`，其主要流程为：

1. 读取 CSV 文件
2. 校验数据合法性
3. 计算加权总分
4. 输出候选点排名
5. 生成三张分析图表

## 9. 方法特点与适用性

本项目所采用的方法具有以下特点：

- 简单直观，适合初学者（比如这个仓库的主人）理解和实现
- 易于修改评分和权重，方便后续扩展
- 能同时兼顾服务效率、安全风险和运行可行性
- 适合作为校园无人机选址的初步分析方法

需要说明的是，本项目主要是一个基于人工评分的综合评价模型，适合用于教学演示和初步比较分析。若用于更复杂的实际部署，还可以进一步加入地图数据、实测距离、GIS 分析、飞行规则约束等更细化的方法。

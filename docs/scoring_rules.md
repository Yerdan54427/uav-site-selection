# 评分细则

本文档给出校园无人机起降点选址分析项目的评分规则，可直接用于 README、课程报告、PPT 或评分说明。

本项目统一采用 **1–5 分制**：

- **5分**：表现优秀
- **4分**：表现较好
- **3分**：表现一般
- **2分**：表现较差
- **1分**：表现很差，但仍纳入候选比较范围

在正式评分前，建议增加如下说明：

**若某点存在明显禁飞风险、场地严重不足、紧邻超高密集人群区等硬约束问题，则不进入候选点评分环节。**

## 一、服务区距离

### 指标含义

候选起降点到主要服务区的距离。服务区可以是宿舍区、教学区、综合服务区等实际配送服务对象较集中的区域。

### 评分逻辑

越近越好。因为离主要服务区越近，用户取件或接收服务越方便，末端步行距离越短，整体服务体验越好。

### 评分标准

| 距离范围 | 分值 | 说明 |
| --- | ---: | --- |
| ≤100 m | 5 | 非常接近主要服务区，服务便利性最高 |
| 100–200 m | 4 | 较接近主要服务区，步行或接收服务较方便 |
| 200–300 m | 3 | 距离适中，仍可接受 |
| 300–400 m | 2 | 距离偏远，服务便利性较差 |
| >400 m | 1 | 距离较远，不利于服务区覆盖 |

## 二、物流中心距离

### 指标含义

候选起降点到校内物流中心的距离。

### 评分逻辑

越近越好。因为距离物流中心越近，装卸衔接越方便，配送起始效率越高。

### 评分标准

| 距离范围 | 分值 | 说明 |
| --- | ---: | --- |
| ≤150 m | 5 | 与物流中心衔接非常高效 |
| 150–250 m | 4 | 衔接较方便，运输效率较高 |
| 250–350 m | 3 | 距离适中，可满足基本需求 |
| 350–500 m | 2 | 距离偏远，衔接效率较低 |
| >500 m | 1 | 距离较远，不利于快速配送 |

## 三、场地开阔度

### 指标含义

候选点周边地面和上空的开阔程度，包括是否有足够起降空间、是否存在明显遮挡。

### 评分逻辑

越开阔越好。因为无人机在起飞和降落阶段对周边空间要求高，场地越开阔，安全裕度越大。

### 评分标准

| 场地情况 | 分值 | 说明 |
| --- | ---: | --- |
| 四周及上空非常开阔，几乎无明显遮挡 | 5 | 非常适合起降 |
| 场地较开阔，仅有少量低风险遮挡 | 4 | 较适合起降 |
| 开阔度一般，存在一定遮挡 | 3 | 基本可用，但需注意起降操作 |
| 场地较局促，周边遮挡较多 | 2 | 起降受限，安全裕度不足 |
| 场地明显狭窄，起降空间不足 | 1 | 不利于安全起降 |

### 可参考观察对象

- 周边是否有高树
- 是否有灯杆、旗杆、宣传牌
- 是否被楼体包围
- 上空是否通透

## 四、障碍物风险

### 指标含义

候选点周边障碍物对无人机飞行、起降和航线安全的影响程度。

### 评分逻辑

障碍物越少、风险越低，分数越高。

### 评分标准

| 障碍物情况 | 分值 | 说明 |
| --- | ---: | --- |
| 周边障碍物极少，飞行环境理想 | 5 | 风险很低 |
| 障碍物较少，对飞行影响较小 | 4 | 风险较低 |
| 存在一定障碍物，需要适当规避 | 3 | 风险中等 |
| 障碍物较多，对起降或航线影响明显 | 2 | 风险较高 |
| 障碍物密集，飞行冲突风险大 | 1 | 风险很高 |

### 可纳入障碍物类型

- 树木、树冠
- 楼体边缘
- 路灯、监控杆
- 架空线缆
- 雕塑、棚架、广告设施

## 五、人流风险

### 指标含义

候选点周边人流密度及其对安全运行的影响。

### 评分逻辑

人流越少越好。因为人流越密集，无人机起降时对人员安全的潜在风险越高。

### 评分标准

| 人流情况 | 分值 | 说明 |
| --- | ---: | --- |
| 平时人流很少，环境较为空旷 | 5 | 人员风险最低 |
| 人流较少，仅偶尔有人经过 | 4 | 风险较低 |
| 人流中等，时段性有人流经过 | 3 | 风险一般 |
| 人流较多，周边常有学生通行 | 2 | 风险较高 |
| 人流密集，属于高频通行区域 | 1 | 风险很高 |

### 建议观察时段

建议不要只看一个时刻，应综合以下时段进行观察：

- 上课前后
- 午餐晚餐高峰
- 晚间活动高峰

这样评分更有说服力。

## 六、航线通达性

### 指标含义

从物流中心到候选点的飞行路线是否顺畅，是否需要明显绕行，是否容易避开建筑和高风险区域。

### 评分逻辑

航线越直接、越顺畅越好。

### 评分标准

| 航线条件 | 分值 | 说明 |
| --- | ---: | --- |
| 航线基本直达，几乎无需绕行 | 5 | 通达性最好 |
| 航线较顺畅，仅需少量避让 | 4 | 通达性较好 |
| 航线一般，存在一定绕行或规避需求 | 3 | 基本可行 |
| 航线较复杂，需要明显绕行 | 2 | 通达性较差 |
| 航线受阻严重，飞行组织困难 | 1 | 通达性很差 |

### 判断时可参考

- 物流中心与候选点之间是否有高楼密集带
- 是否需要绕开大面积人群区
- 是否容易设计出稳定、重复的固定航线

## 七、运维便利性

### 指标含义

候选点在后续建设、管理、值守、维护和日常运行方面的便利程度。

### 评分逻辑

越容易建设和管理，分数越高。

### 评分标准

| 运维条件 | 分值 | 说明 |
| --- | ---: | --- |
| 可直接利用，建设简单，管理方便 | 5 | 运维最便利 |
| 略需调整，整体实施较方便 | 4 | 运维较便利 |
| 需一定改造，管理条件一般 | 3 | 基本可实施 |
| 改造需求较多，管理不够方便 | 2 | 运维较困难 |
| 实施和管理都较困难 | 1 | 运维很不便利 |

### 可参考内容

- 地面是否平整
- 是否便于设置标识、围挡、起降区
- 是否方便工作人员到达
- 是否便于巡检和日常管理
